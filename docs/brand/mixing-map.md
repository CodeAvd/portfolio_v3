# Mixing Map

## Source breakdown

| Layer | Source direction | Applied decision |
| --- | --- | --- |
| Layout | Swiss / systems | strict section framing, ledger grids, evidence-board hero |
| Typography | editorial publishing | `Newsreader` display tone with sharp hierarchy |
| UI body + metadata | developer / systems | `Public Sans` + `IBM Plex Mono` for dense, readable proof |
| Color | industrial field manual | bone, graphite, signal sage, oxide, amber |
| Components | support tooling / dashboards | chips, rails, dossier cards, utility bars |
| Motion | instrument UI | short reveals, no drifting ambience |

## Uniqueness check

The redesign passes if it does **not** read as:

1. A luxury personal site
2. A generic SaaS portfolio
3. A crypto/web3 operator page

## Applied synthesis

- The hero is no longer a portrait stage; it is a signal board with proof, operating lanes, and a smaller portrait trust cue.
- Navigation, cards, and utility controls now read like dossier surfaces instead of glossy product chrome.
- The wordmark and monogram use a routing cut to suggest triage and handoff rather than decorative branding.
