# Signal Ledger Brand Package

This folder contains the visual identity deliverables for the `portfolio_true` refresh.

## Contents

- `research.md`: six-layer audit, competitor matrix, and moodboard direction
- `mixing-map.md`: the exact layer-by-layer synthesis used for the redesign
- `style-guide.md`: palette, typography, logo, component, and motion rules

## Implemented outputs

- Shared token source: `src/content/design-tokens.json`
- Shared brand metadata: `src/content/brand.ts`
- Reusable mark and wordmark assets: `public/brand/`
- Runtime brand shell: home, case study, resume, OG image, icon, and favicon

## Core positioning

- Concept: `Signal Ledger`
- Audience: support ops and technical support hiring
- Character: forensic, composed, operator-grade
- Visual rule: proof leads, portrait supports
