# Resume Generation Prompt

Design a resume page as a premium hiring dossier.

The page must feel:

- serious
- printable
- direct
- evidence-based

The page is not a second marketing homepage.

Required sections:

1. identity header
2. recruiter snapshot
3. evidence highlights
4. experience
5. selected proof
6. skills
7. education and languages

Rules:

- use stronger information hierarchy than decoration
- keep key hiring facts visible without scrolling too far
- make print mode clean and credible
- let the visuals support trust, not compete with it

The page should still feel premium, but the dominant emotion is confidence, not spectacle.
