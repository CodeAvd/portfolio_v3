# Prompt Pack

This folder holds the prompt system for the premium portfolio direction.

- `concept-brief.md`: creative direction and site role.
- `home-generation.md`: generation prompt for the homepage.
- `resume-generation.md`: generation prompt for the resume page.
- `critique-rubric.md`: scoring system for review.
- `revision-prompt.md`: rewrite prompt for the next iteration.

Use these prompts to judge the site as both a showcase and a recruiter trust document.
