# Home Generation Prompt

Create a homepage for a premium portfolio that behaves as both:

- a visually memorable showcase
- a recruiter-ready trust summary

Audience:

- recruiters
- hiring managers
- support leaders
- product-minded operators reviewing the profile quickly

The page must communicate within the first screen:

- who the profile is for
- what the person does
- why the profile is different
- what the next step is

Required structure:

1. Hero thesis with strong editorial visual treatment.
2. Primary CTA to resume.
3. Secondary CTA to contact.
4. Recruiter snapshot with trust metrics.
5. One flagship case chapter.
6. Compact proof appendix for remaining cases.
7. Resume preview / role fit block.
8. Final contact CTA.

Required content goals:

- show support operations and technical support credibility
- prove escalation and workflow thinking
- show builder evidence without turning the site into an engineer portfolio
- keep the copy scannable for recruiters

Motion rules:

- one strong hero sequence
- one additional scroll chapter
- reduced-motion fallback must preserve all meaning

Do not generate a generic landing page. The result must feel premium, sharp, and authored.
