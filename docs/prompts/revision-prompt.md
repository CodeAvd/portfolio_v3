# Revision Prompt

Revise the portfolio using the critique results.

Rules for the revision:

- preserve the premium editorial cinema direction
- improve recruiter clarity before adding more visual complexity
- keep the CTA hierarchy intact
- remove any decorative element that weakens proof or scanning
- strengthen proof signals before adding new animation

When revising, explicitly state:

1. what changed
2. why it improves the rubric score
3. which recruiter concern it resolves
4. which visual decision remains intentional and should not be simplified
